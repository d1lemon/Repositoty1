# Repositoty1

This is a repository for artificial intelligence, machine learning and data analysis. Programs are in Python and R. 
The Python programs include libraries such as matplotlib, Pandas, NumPy, Seaborn, Scikit Learn, PySpark, and TensorFlow. 
The datasets used are all in the public domain. The analysis include, but not limited to, Logistic regression, Decission 
tree classification, clustering, random forest, gradient boosting, apriori algorithm, etc.
